# disappearing_rooms
a place where persons can join a temporary room and chat.

## starting the server
```bash
npm run devStart
```

## using application

type this url
```sh
localhost:3000
```
![roomInterface](./pics/firstPerson.png)

enter the room name and your nickname to join that room
